Sanara is made with various services, you are free to contribute to any of them by opening an issue or doing a pull request:
  • Sanara: The bot that you interact with on Discord
  • Sanara-Website: This website
  • Bots-db: Backend that Sanara use to send her stats to this website
  • ProgramManager: Backend that automatically restart the bot when it is down
  • admin.zirk.eu: Front-end of ProgramManager
  • Discord-Utils: Utilities functions used by the Discord bot

Using Sanara by cloning the repository▼

At first you'll need to clone Sanara repository recursively:
git clone --recurse-submodules https://github.com/Xwilarg/Sanara.git

Then launch install.sh, it'll download the following:
  • ReThinkdb (database used by Sanara)
  • All files needed for game module
  • Opus, libsodium, ffmpeg and youtube-dl for radio module
Once the script is done, go in SanaraV2/bin/Debug/Keys and create a file named "Credentials.json", this file will contains all your keys in the JSON format:
"botToken" : "AAABBBCCC",
There is one mandatory value:
  • BotToken: Your bot token (you can obtain it on the Discord developper portail)
All the others fields are optional:
  • WebsiteStatsUrl: Contains the website in which Sanara's stats are sent
  • WebsiteStatsToken: Authentification token for sending stats
  • UploadWebsiteLocation: Local path to the files of the website that the download command use (used to upload ZIP files)
  • UploadWebsiteUrl: URL of the website that will upload files
  • YoutubeKey: YouTube API key
Once it's done, launch Visual Studio and start the project.
